All images are loaded via Google Drive links

How to get a Google Drive image link
  1. Upload your image (logo or background photo) to Google Drive
  2. Right-click the file → Share → set to "Anyone with the link"
  3. Click Copy link → paste it in the field below
  4. The URL is automatically converted — you don't need to edit it

Setup Guide

How to use this dashboard system

Files Required

Keep these 3 files in the same folder:

  • index.html — open this on the TV (press F11 for fullscreen)
  • admin.html — open this on the organizer laptop
  • data.js — shared data file, do not edit directly
Live Updates

Both pages share browser localStorage. Changes saved here appear on the display within 30 seconds automatically — or instantly when the slide changes.

Important: Admin and display must be open in the same browser on the same computer for localStorage to sync. For multi-device use, Google Sheets integration can be added.
